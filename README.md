# Projeto: Tabuada

Este projeto é uma aplicação web simples criada com **HTML**, **CSS** e **JavaScript** que permite ao usuário gerar a tabuada de um número.

## 📌 Funcionalidades

- Entrada de um número inteiro
- Geração automática da tabuada de 1 a 10
- Exibição dos resultados em uma lista

## 💡 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (puro)
